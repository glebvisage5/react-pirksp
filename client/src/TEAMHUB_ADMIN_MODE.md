# TeamHub Admin Mode - Документация

## Обзор

Admin Mode в TeamHub — это локальная админская панель для управления всеми сущностями сервиса TeamHub. Он **НЕ заменяет** глобальный Admin Center и **НЕ влияет** на EduCRM.

## Доступ к Admin Mode

1. Авторизуйтесь как администратор (isAdmin = true)
2. В TeamHub появится переключатель "Mode" в верхней панели
3. Переключите режим с "User" на "Admin"

## Структура Admin Mode

### 9 модулей администрирования:

#### 1. **Dashboard TeamHub**
- Общая статистика по всем командам
- Количество участников, проектов, задач, ТЗ
- Активность за сутки/неделю
- Последние действия
- Системные уведомления

#### 2. **Команды (Teams)**
- Просмотр всех команд
- Создание/редактирование/удаление команды
- Управление владельцами
- Просмотр активности команды
- Перенос участников между командами

#### 3. **Участники (Members)**
- Просмотр всех участников TeamHub
- Приглашение новых участников
- Редактирование профилей
- Назначение ролей
- Перенос участников между командами
- Просмотр задач участника

#### 4. **Роли TeamHub (Roles)**
- Управление локальными ролями TeamHub
- Создание кастомных ролей
- Настройка прав доступа:
  - Управление участниками
  - Управление проектами
  - Управление задачами
  - Управление ТЗ
  - Управление файлами
  - Просмотр статистики

**Роли по умолчанию:**
- Team Leader (полный доступ)
- Moderator (управление проектами, задачами, ТЗ)
- Developer (работа с задачами и ТЗ)
- Designer (дизайн, ТЗ, файлы)
- Viewer (только просмотр)

#### 5. **Проекты (Projects)**
- Просмотр всех проектов
- Создание/редактирование/удаление проектов
- Управление привязкой к командам
- Управление ТЗ и задачами
- Просмотр прогресса

#### 6. **Задачи (Tasks)**
- Просмотр всех задач TeamHub
- Фильтры по команде, проекту, участнику, статусу
- Создание/редактирование/удаление задач
- Массовое назначение задач
- Статистика выполнения

#### 7. **Технические задания (ТЗ)**
**Ключевой модуль** с 3 вкладками:
- **Viewer** - просмотр ТЗ в красивом веб-формате
- **Builder** - блочный редактор (как Notion)
- **Versions** - управление версиями

**Типы блоков:**
- Заголовки (H1, H2, H3)
- Текст
- Списки (маркированные, нумерованные)
- Таблицы
- Изображения
- Код
- Ссылки
- Чеклисты

**Возможности:**
- Drag & Drop блоков
- Экспорт в PDF
- Версионирование
- Привязка к проектам
- Создание задач из пунктов ТЗ

#### 8. **Файлы (Files)**
- Просмотр всех файлов TeamHub
- Загрузка/удаление файлов
- Создание папок
- Настройка видимости (команда, проект)
- Статистика загрузок

#### 9. **Настройки TeamHub (Settings)**
Настройки:
- **Общие:** тема, язык
- **Уведомления:** email, push
- **Задачи:** автоназначение, оценка времени
- **ТЗ:** версионирование, автосохранение
- **Файлы:** лимиты размера, квота хранилища
- **Роли:** кастомные роли, иерархия
- **Интеграции:** Telegram-бот, webhooks, API

## Отличия от обычного режима

| Функция | User Mode | Admin Mode |
|---------|-----------|------------|
| Доступ к командам | Только свои | Все команды |
| Управление участниками | Нет | Полное |
| Управление ролями | Нет | Полное |
| Просмотр статистики | Своя команда | Вся система |
| Удаление сущностей | Ограничено | Полное |
| Настройки системы | Нет | Полный доступ |

- Отдельное меню навигации для админ-режима
- Состояние `teamHubMode: "user" | "admin"`
- Условный рендеринг контента

### UI особенности:
- Badge "Admin" на всех админ-страницах
- Красный цвет переключателя в админ-режиме
- Расширенная статистика на каждой странице
- Mock данные для демонстрации

## Важно!

1. Admin Mode работает **только внутри TeamHub**
2. Не влияет на EduCRM и другие сервисы
3. Требует роль `isAdmin = true`
4. Локальная админ-панель, не глобальная
5. Все изменения применяются только к TeamHub
