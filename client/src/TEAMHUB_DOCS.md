# TeamHub - Документация

## Обзор
TeamHub - это полноценный сервис для командной работы, интегрированный в экосистему InnovateCRM. Сервис включает управление командами, проектами, задачами и конструктор технических заданий.

## Структура проекта

```
/components/teamhub/
├── TeamHub.tsx              # Основной компонент с навигацией
├── TeamHubDashboard.tsx     # Главная панель управления
├── Teams.tsx                # Управление командами
├── Projects.tsx             # Управление проектами
├── Tasks.tsx                # Канбан-доска для задач
├── SpecBuilder.tsx          # Конструктор ТЗ (блочный редактор)
├── SpecViewer.tsx           # Просмотр ТЗ с оглавлением
├── SpecVersions.tsx         # Управление версиями ТЗ
└── Members.tsx              # Управление участниками
```

## Модули TeamHub

### 1. TeamHub Dashboard
**Файл:** `TeamHubDashboard.tsx`

**Функционал:**
- Быстрая статистика (команды, проекты, задачи, ТЗ)
- Быстрые действия (создание команды, проекта, ТЗ, приглашение участников)
- Лента недавней активности

**Особенности:**
- Адаптивная сетка статистики (1-4 колонки)
- Цветовые градиенты для каждой категории
- Интерактивные карточки с hover эффектами

### 2. Teams (Команды)
**Файл:** `Teams.tsx`

**Функционал:**
- Создание новых команд
- Поиск по командам
- Просмотр информации о команде
- Управление участниками (только для Team Leader)
- Роли: Team Leader, Moderator, Member, Viewer

**Возможности:**
- Каждый пользователь может создавать несколько команд
- Разные роли в разных командах
- Приглашение новых участников
- Выход из команды (кроме владельца)

### 3. Projects (Проекты)
**Файл:** `Projects.tsx`

**Функционал:**
- Создание проектов
- Фильтрация по статусу (Planning, Active, Completed, On Hold)
- Поиск проектов
- Прогресс-бар выполнения
- Связь с командами и ТЗ
- Дедлайны и отслеживание сроков

**Статусы проектов:**
- Planning - планирование
- Active - активный
- Completed - завершён
- On Hold - на паузе

### 4. Tasks (Задачи)
**Файл:** `Tasks.tsx`

**Функционал:**
- Канбан-доска с 4 колонками (To Do, In Progress, Review, Done)
- Создание задач
- Приоритеты (High, Medium, Low)
- Назначение исполнителей
- Связь с проектами
- Создание задач из блоков ТЗ

**Статусы задач:**
- To Do - к выполнению
- In Progress - в работе
- Review - на проверке
- Done - выполнено

### 5. Spec Builder (Конструктор ТЗ)
**Файл:** `SpecBuilder.tsx`

**Основная функция:** Блочный редактор для создания структурированных технических заданий.

**Типы блоков:**
1. **Heading** - заголовки (H1, H2, H3)
2. **Text** - текстовые блоки
3. **List** - списки (нумерованные и маркированные)
4. **Table** - таблицы
5. **Image** - изображения
6. **Code** - блоки кода (с подсветкой синтаксиса)
7. **Link** - ссылки
8. **Checklist** - чеклисты (с возможностью создания задач)

**Возможности:**
- Drag & Drop для перемещения блоков
- Добавление/удаление блоков
- Перемещение блоков вверх/вниз
- Автосохранение
- Создание задач из пунктов чеклиста
- Многоязычность

**Пример использования:**
```typescript
// Создание блока заголовка
{
  type: "heading",
  content: { text: "Project Overview", level: 1 }
}

// Создание блока списка
{
  type: "list",
  content: { 
    items: ["Item 1", "Item 2"], 
    ordered: false 
  }
}

// Создание блока кода
{
  type: "code",
  content: { 
    code: "console.log('Hello')", 
    language: "javascript" 
  }
}
```

### 6. Spec Viewer (Просмотр ТЗ)
**Файл:** `SpecViewer.tsx`

**Функционал:**
- Красивый просмотр ТЗ в браузере
- Оглавление с навигацией
- Плавная прокрутка к разделам
- Информация о версии, авторе, команде
- Экспорт в PDF
- Возможность поделиться ссылкой
- Переход в режим редактирования

**Особенности:**
- Адаптивная верстка
- Sticky боковая панель с оглавлением
- Подсветка активного раздела
- Поддержка тёмной/светлой темы

### 7. Spec Versions (Версии ТЗ)
**Файл:** `SpecVersions.tsx`

**Функционал:**
- История всех версий ТЗ
- Временная шкала изменений
- Просмотр изменений в каждой версии
- Восстановление предыдущих версий
- Сравнение версий
- Комментарии к версиям

**Информация о версии:**
- Номер версии
- Дата и время создания
- Автор изменений
- Комментарий
- Список изменений
- Статус (текущая/архивная)

### 8. Members (Участники)
**Файл:** `Members.tsx`

**Функционал:**
- Управление участниками команды
- Приглашение новых участников
- Назначение ролей
- Отслеживание приглашений
- Поиск участников
- Удаление участников

**Вкладки:**
- All Members - все участники
- Invitations - приглашения (pending, accepted, declined)

**Роли:**
- Team Leader - полный доступ
- Moderator - управление задачами и ТЗ
- Member - обычный участник
- Viewer - только просмотр

## Интеграция в App.tsx

```typescript
// 1. Импорт компонента
import { TeamHub } from "./components/teamhub/TeamHub";

// 2. Добавление в AppView
type AppView = "company" | "landing" | "dashboard" | "admin-center" | "teamhub";

// 3. Обработка навигации
const handleServiceSelect = (serviceId: string) => {
  if (serviceId === "teamhub") {
    setAppView("teamhub");
  }
};

// 4. Рендеринг
if (appView === "teamhub") {
  return (
    <TeamHub
      onBackToServices={handleBackToCompany}
      onLogout={handleLogout}
    />
  );
}
```

## Дизайн-система

### Цветовая схема
- Основной градиент: `from-emerald-500 to-teal-600`
- Текст заголовков: `from-emerald-600 to-teal-600`
- Тёмная тема: `from-emerald-400 to-teal-400`

### Иконки
- Используется библиотека lucide-react
- Размеры: h-4 w-4 (маленькие), h-5 w-5 (средние), h-6 w-6 (большие)

### Компоненты UI
Все компоненты используют shadcn/ui:
- Button
- Card
- Input, Textarea
- Select, Dialog
- Badge, Tabs
- DropdownMenu
- ScrollArea, Separator

## Многоязычность

Поддерживаются языки:
- English (en)
- Русский (ru)

Все тексты переводятся динамически через объект `t`:

```typescript
const t = {
  teams: language === "en" ? "Teams" : "Команды",
  createTeam: language === "en" ? "Create Team" : "Создать команду",
  // ...
};
```

## Адаптивность

Все компоненты адаптивны для:
- Mobile (< 768px)
- Tablet (768px - 1024px)
- Desktop (> 1024px)

Используются Tailwind классы:
- `sm:` - от 640px
- `md:` - от 768px
- `lg:` - от 1024px
