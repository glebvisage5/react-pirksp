# TeamHub - Краткое резюме обновления навигации

## Что было изменено

Полностью переработана структура навигации TeamHub в соответствии с требованиями пользователя.

## Основные изменения

### 1. Новое главное меню (5 вкладок)

**Вместо старых 8 вкладок:**
- Dashboard, Teams, Projects, Tasks, Spec Builder, Spec Viewer, Spec Versions, Members

**Теперь 5 вкладок:**
1. ✅ **Панель управления** (Dashboard)
2. ✅ **Команды** (Teams)
3. ✅ **Дедлайны** (Deadlines) - **НОВЫЙ**
4. ✅ **Профиль** (Profile) - **НОВЫЙ**
5. ✅ **Настройки** (Settings) - **НОВЫЙ**

### 2. Динамическое меню команды

При выборе команды **левое меню полностью меняется** и показывает вкладки команды:

- ⬅️ Кнопка "Назад к командам"
- 📊 Обзор
- 📁 Проекты
- 📄 Технические задания
- ✅ Задачи
- 👥 Участники
- 📂 Файлы
- 🛡️ Роли команды (только Team Leader и Moderator)
- ⚙️ Настройки команды (только Team Leader и Moderator)

### 3. Новые компоненты

#### `/components/teamhub/Deadlines.tsx`
- Все дедлайны из всех команд
- Фильтрация: Все, Сегодня, Предстоящие, Просрочено, Завершено
- Приоритеты: Высокий, Средний, Низкий
- Поиск по названию, команде, проекту

#### `/components/teamhub/Profile.tsx`
- Личная информация пользователя
- Редактирование профиля
- Загрузка аватара
- Статистика: команды, проекты, выполненные задачи

#### `/components/teamhub/Settings.tsx`
- Внешний вид: тема (светлая/темная/системная), язык (EN/RU)
- Уведомления: Email, Push, обновления задач, приглашения в команды, обновления проектов, еженедельная сводка
- Конфиденциальность и безопасность

### 4. Обновленные компоненты

#### `/components/teamhub/TeamHub.tsx`
- Новая логика навигации с поддержкой MainView и TeamView
- Динамическое переключение меню при выборе команды
- Состояние selectedTeam для отслеживания текущей команды
- Callbacks для выбора команды и возврата к списку команд

#### `/components/teamhub/Teams.tsx`
- Убрана внутренняя навигация в TeamView
- Добавлен callback `onTeamSelect` для передачи выбранной команды наверх
- Компонент теперь только отображает список команд

#### `/components/teamhub/TeamHubDashboard.tsx`
- Обновлены Quick Actions для работы с новыми вкладками
- Изменен тип навигации на MainView

#### `/components/teamhub/team-views/TeamSpecs.tsx`
- Удален неиспользуемый параметр `onCreateSpec`

### Навигация

1. **Основной уровень**: Dashboard → Teams → Deadlines → Profile → Settings
2. **Уровень команды**: Overview → Projects → Specs → Tasks → Members → Files → Roles → Team Settings
3. **Переход между уровнями**: Выбор команды / Кнопка "Назад к командам"

## Ролевая система

Вкладки команды автоматически фильтруются по роли:
- **Team Leader** и **Moderator**: доступны все вкладки
- **Member** и **Viewer**: скрыты "Роли команды" и "Настройки команды"

## Как использовать

1. Откройте TeamHub из главного дашборда
2. Используйте главное меню для навигации по основным разделам
3. Выберите команду из списка "Команды"
4. Левое меню автоматически изменится на меню команды
5. Используйте кнопку "Назад к командам" для возврата
